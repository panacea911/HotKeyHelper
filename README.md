# 🎹 HotKeyHelper

**HotKeyHelper** — это легкое, быстрое и функциональное Windows-приложение на базе **.NET 8.0 (Windows Forms)** для создания и настройки глобальных горячих клавиш. Утилита работает в системном трее, поддерживает автозапуск и позволяет автоматизировать повседневные задачи одной комбинацией клавиш.

---

## ✨ Основные возможности

* 🚀 **Запуск приложений и открытие папок**: Быстрый запуск программ и папок по хоткеям.
* 🖥️ **Управление окнами**:
  * Сворачивание, разворачивание/восстановление активного окна.
  * Мягкое закрытие активного окна или принудительное завершение процесса (Kill).
  * Фокусировка: Умная и надежная система перехвата фокуса ввода для запускаемых приложений.
* 🔊 **Мультимедиа**: Изменение громкости звука (+/-), мьют, управление воспроизведением (плей/пауза).
* 🔒 **Безопасность**: Быстрая блокировка экрана Windows (`Win + L` аналог).
* ⚙️ **Удобный интерфейс настроек**: Назначение клавиш через окно перехвата кликов (Catch Form), редактирование параметров и аргументов командной строки.
* 📦 **Интеграция с Windows**: Опция запуска вместе со стартом системы (автозагрузка через ярлыки).

---

## 🛠️ Поддерживаемые действия

| Название действия | Что делает |
| :--- | :--- |
| **Запустить программу** | Запуск любого `.exe`, `.bat` или скрипта с передачей аргументов и гарантированной активацией окна. |
| **Открыть папку** | Быстрое открытие нужного каталога в Проводнике. |
| **Закрыть активное окно (Мягко)** | Отправляет системный сигнал закрытия окна (`WM_CLOSE`). |
| **Завершить процесс окна** | Принудительно «убивает» процесс, которому принадлежит текущее активное окно. |
| **Развернуть / Восстановить окно** | Переключает режим отображения окна (на весь экран / оконный). |
| **Свернуть окно** | Мгновенно сворачивает текущее активное окно. |
| **Заблокировать экран Windows** | Вызывает блокировку рабочей станции. |
| **Управление громкостью и медиа** | Увеличение/уменьшение громкости, включение/выключение звука, пауза/воспроизведение треков. |

## 🚀 Как запустить и собрать проект

### Системные требования
* Операционная система: **Windows 10 / 11**
* Установленный **.NET 8.0 Runtime** (или .NET 8.0 SDK для сборки).

### Сборка из исходников
Для сборки приложения вручную выполните в папке проекта:
```bash
dotnet publish -c Release -r win-x64 --self-contained false -p:PublishSingleFile=true -o ./publish_output
```
После завершения в папке `publish_output` появится единственный исполняемый файл `HotKeyHelper.exe`.

## 🤖 Автоматическая сборка (CI/CD GitHub Actions)

В проекте настроены два воркфлоу для автоматизации:
1. **Тестовая автосборка** (`test` ветка): При любом коммите в ветку `test` собирается тестовый билд, доступный в артефактах GitHub Actions.
2. **Публикация релизов** (по тегу `v*`): При отправке тега, начинающегося с буквы `v` (например, `v1.0.3`), автоматически создается релиз на GitHub с прикрепленным собранным архивом `HotKeyHelper-win-x64.zip`.
